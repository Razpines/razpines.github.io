# razpines.github.io

Portfolio + projects + blog, published with GitHub Pages.

## Publish

1. Push to `main`
2. In GitHub: **Settings -> Pages**
3. Source: **Deploy from a branch**
4. Branch: `main` / folder: `/ (root)`

## Run locally (optional)

If you have Ruby + Bundler:

```sh
bundle exec jekyll serve
```
